# Ejercicio 3: Gestión de Estudiantes

## Descripción
Simulación de un pequeño sistema de base de datos de estudiantes utilizando métodos de búsqueda y actualización.

## Métodos Clave Utilizados
* **`add(Object)`**: Para registrar estudiantes manualmente.
* **`contains(Object)`**: Realiza una búsqueda lineal para verificar si un estudiante existe en la lista. Retorna un booleano (`true`/`false`).
* **`remove(Object)`**: Elimina la primera ocurrencia del objeto especificado. A diferencia de `remove(int index)`, este método busca por valor.

## Notas
El código incluye una sección comentada para habilitar la entrada por teclado (`Scanner`) si se desea hacer el programa interactivo.
