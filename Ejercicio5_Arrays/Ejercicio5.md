# Ejercicio 5: Mini Inventario

## Descripción
Este es el desafío final que integra múltiples operaciones de manipulación de listas para simular un control de stock básico.

## Características Implementadas
1.  **Almacenamiento:** Uso de `ArrayList<String>` para guardar nombres de productos.
2.  **Gestión:** Capacidad de agregar productos y eliminar productos específicos por nombre.
3.  **Ordenamiento:**
    * Se utiliza la clase utilitaria `java.util.Collections`.
    * El método `Collections.sort(lista)` ordena los elementos (Strings) en orden alfabético (lexicográfico) ascendente.
4.  **Conteo:** Reporte final de la cantidad de items en stock usando `.size()`.
