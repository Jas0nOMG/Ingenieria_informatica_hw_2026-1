# Ejercicio 1: ArrayList Básico

## Descripción
Este programa demuestra cómo crear y manipular un `ArrayList` básico en Java.

## Funcionalidad
1. **Creación:** Se inicializa un `ArrayList<String>` llamado `frutas`.
2. **Inserción:** Se agregan 5 elementos (frutas) usando el método `.add()`.
3. **Tamaño:** Se utiliza el método `.size()` para mostrar la cantidad total de elementos.
4. **Iteración:** Se muestran dos formas de recorrer la lista:
   - **Bucle For tradicional:** Accediendo a los elementos por su índice mediante `.get(i)`.
   - **Bucle For-Each:** Iterando directamente sobre los objetos de la colección.

## Salida Esperada
El programa imprimirá el número 5 seguido de la lista de frutas dos veces (una por cada tipo de bucle).
