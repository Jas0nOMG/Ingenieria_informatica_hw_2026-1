## 1. Enunciado del Ejercicio

> The following program randomly generates an integer greater than or equal to 0 and less than 100. The program then prompts the user to guess the number. If the user guesses the number correctly, the program outputs an appropriate message. Otherwise, the program checks whether the guessed number is less than the random number. If the guessed number is less than the random the number generated by the program, the program outputs the message, ‘‘Your guess is lower than the number’’; otherwise, the program outputs the message, ‘‘Your guess is higher than the number’’. The program then prompts the user to enter another number. The user is prompted to guess the random number until the user enters the correct number.

## 2. Análisis del Problema

El objetivo es crear un juego de consola donde la computadora elige un número secreto y el usuario intenta adivinarlo mediante pistas.

### Entradas (Inputs)
* **Números enteros:** Ingresados por el usuario en cada intento de adivinanza.

### Salidas (Outputs)
* **Mensaje de bienvenida:** Instrucciones iniciales.
* **Pistas de rango:**
    * "Your guess is lower than the number" (Si el número ingresado es menor al secreto).
    * "Your guess is higher than the number" (Si el número ingresado es mayor al secreto).
* **Mensaje de éxito:** Confirmación cuando el usuario acierta.

### Variables a Emplear

| Variable | Tipo de Dato | Descripción |
| :--- | :--- | :--- |
| `randomNum` | `int` | Almacena el número aleatorio generado por el sistema (0-99). |
| `guess` | `int` | Almacena el número ingresado por el usuario en cada intento. |
| `input` | `Scanner` | Objeto necesario para leer la entrada del teclado. |

## 3. Diseño del Algoritmo

Para resolver este problema se utilizará una estructura de control iterativa (bucle `while`) que se ejecutará indefinidamente hasta que el usuario acierte el número.

### Pseudocódigo

1.  **Inicio**
2.  Generar `randomNum` entre 0 y 99 usando `Math.random()`.
3.  Imprimir "Guess the number between 0-100".
4.  **Mientras** (`true`):
    5.  Leer `guess` del usuario.
    6.  **Si** (`guess` == `randomNum`):
        * Imprimir "You won!".
        * **Romper** el ciclo (break).
    7.  **Si no, Si** (`guess` < `randomNum`):
        * Imprimir "Your guess is lower than the number".
        * Imprimir "Make another guess".
    8.  **Si no** (`guess` > `randomNum`):
        * Imprimir "Your guess is higher than the number".
        * Imprimir "Make another guess".
9.  **Fin**
